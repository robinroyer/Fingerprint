# Browser Fingerprint Scanner

[![Try the fingerprint scanner](https://img.shields.io/badge/Try%20it%20live-fingerprint--948z.vercel.app-00d4ff?style=for-the-badge)](https://fingerprint-948z.vercel.app/)

![Fingerprint Scanner](screenshot.png)

A browser fingerprinting demo that collects and displays device, browser, and behavioral signals — all without cookies or login.

It runs entirely client-side and reveals how much information a website can silently gather about your device.




## What it detects

- **Browser** — name, version, engine, user agent
- **Screen** — resolution, device pixel ratio, color depth
- **Hardware** — CPU cores, memory, GPU renderer, touch support
- **Locale** — timezone, language, DST offset
- **Network** — connection type, downlink speed, local IPs (WebRTC)
- **Performance** — JS heap usage, CPU timing
- **Storage** — cookie availability
- **Canvas fingerprint** — unique hash from rendered graphics
